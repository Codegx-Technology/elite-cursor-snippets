## 🚀 **Quick Start**

### **Option 1: Clone & Deploy (Recommended)**
```bash
# Clone the arsenal to any project
git clone https://github.com/Codegx-Technology/elite-cursor-snippets
cd elite-cursor-snippets

# Deploy to your project
cp -r .vscode/* /your/project/.vscode/
cp AI-Prompt-Arsenal.md /your/project/
```

### **Option 2: Global Installation**
```bash
# Deploy globally (available in all Cursor workspaces)
./sync-arsenal.sh -g
# or
.\deploy-elite-arsenal.ps1 -Global
```

### **Option 3: Cross-Platform Deployment**
```bash
# Using bash script (macOS/Linux)
./sync-arsenal.sh -p /path/to/your/project

# Using PowerShell script (Windows)
.\deploy-elite-arsenal.ps1 -ProjectPath "C:\path\to\your\project"
```

## 🧬 **Smart Context System**

### **Context Variables**
Use these in your code for better AI understanding:

```javascript
// [TASK]: Current task description
// [GOAL]: Desired outcome
// [CONSTRAINTS]: Limitations/requirements
// [SNIPPET]: Which prompt to use
// [CONTEXT]: Background information
// [PROGRESS]: Current status
// [NEXT]: Next action
// [LOCATION]: File/line reference
```

### **Example Context Chain**
```javascript
// [TASK]: Implement user authentication flow
// [GOAL]: Secure login with JWT tokens
// [SNIPPET]: thinkwithai
// [CONSTRAINTS]: Must work offline, no breaking changes

// Now, let's think through this step by step...
```
